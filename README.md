# flux

## Project structure

`src` - исходники (typescrypt)
`build` - рабочие скрипты (javascript)

`__tests__` - тесты

### Src:

```
src/
    __tests__/
        testEventEmitter.js
        testDispatcher.js
    EventEmitter.js
    Dispatcher.js
```

### Build:

```
build/
    __tests__/
        testEventEmitter.ts
        testDispatcher.ts
    EventEmitter.ts
    Dispatcher.ts
```
