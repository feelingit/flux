# flux

## Структура проекта

`src` - исходники (typescrypt)
`build` - рабочие скрипты (javascript)

`__tests__` - тесты

### src:

```
src/
    __tests__/
        testEventEmitter.js
        testDispatcher.js
    EventEmitter.js
    Dispatcher.js
```

### build:

```
build/
    __tests__/
        testEventEmitter.ts
        testDispatcher.ts
    EventEmitter.ts
    Dispatcher.ts
```
